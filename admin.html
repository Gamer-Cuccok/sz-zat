Közös magyar szólista

Nincs külön tipp-lista és megfejtés-lista. Egy darab canonical fájl van: data/words.json. Minden aktív szó tippelhető, és következő köröktől megfejtésként is kisorsolható.

GitHub beállítások

A GitHub token böngészőben nem való publikus adminhoz. Az admin induláskor a data/words.json mellé automatikusan behúzza a Firebase-ben élesített online szavakat is.

Szó hozzáadás

Egy szó, több szó vagy nagy TXT lista is mehet. A Hozzáadás egyetlen menetben berakja az admin listába, élesíti Firebase-ben, és ha megadtad a GitHub adatokat, menti a data/words.json fájlba is.

Nincs kiválasztott TXT fájl.
